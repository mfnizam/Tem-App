# Tem-App
This Project is for maintaining water on fish ponds. so the water can be healty for the fish.
it use arduino for processing  the sensor and nomcu Esp board to send data to node.js server using mqtt communication. the app can control and monitor all the actuator.

what in this project folder:
Aplikasi - contain the code of Ionic 3 App.
Arduino - contain the code for arduino and nodemcu Esp board
Server - contain node.js server with Mongodb Database.
